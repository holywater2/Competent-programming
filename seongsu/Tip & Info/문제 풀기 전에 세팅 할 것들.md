##1. cin/cout 속도 향상(cin cout setting)


```cpp
#include <iostream>

int main() {
    std::ios::sync_with_stdio(false);
    cin.tie(NULL);
    //위의 구문을 세팅해주면 2~3배 정도 빨라진다.
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
```
